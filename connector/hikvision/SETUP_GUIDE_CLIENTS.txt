================================================================================
  HIKVISION CONNECTOR - CLIENT SETUP (Windows)
  Same idea as ZK connector: install once, runs at startup.
================================================================================

WHAT YOU NEED
-------------
- One Windows PC on the same office network as the Hikvision device
- Files from your provider: connector-hik.exe, config.example.hikvision.json,
  install-windows.bat
- Device IP (Communication -> Ethernet on the terminal)
- Device admin login (usually username: admin, password from installer)
- Device API key from PunchPay: Devices -> your device -> copy API key


STEP 1 - Folder
---------------
Create e.g. C:\AttendanceConnectorHik and copy all files there.


STEP 2 - Config
---------------
Rename config.example.hikvision.json to config.hikvision.json
Open in Notepad. Set:
  deviceIp          = your Hikvision device IP
  hikUsername       = admin (unless your IT said otherwise)
  hikPassword       = device admin password
  deviceApiKey      = from PunchPay Devices page
  backendUrl        = https://punchpay.in  (or URL your provider gave)

Save.


STEP 2b - First test (recommended)
----------------------------------
Double-click run-once-debug.bat (same folder as connector-hik.exe).
You should see messages and get connector-hik.log. If the window shows
a config error, fix config.hikvision.json — it must be in the SAME folder
as connector-hik.exe (if the exe is inside dist\, put config there too).


STEP 3 - Install (Administrator)
--------------------------------
Right-click install-windows.bat -> Run as administrator -> Yes.

This creates a scheduled task "AttendanceConnectorHik" so the connector
starts when Windows starts.

Log file: connector-hik.log in the same folder as connector-hik.exe.


STEP 4 - Verify
---------------
Do a test punch on the device. In PunchPay, check Attendance and Devices
Last sync.


STOP / REMOVE
-------------
Task Scheduler -> find "AttendanceConnectorHik" -> Disable or Delete.


================================================================================
  MAC
================================================================================
Use connector-hik-mac and install-mac.sh from the same folder as
config.hikvision.json. Logs: connector-hik.log
